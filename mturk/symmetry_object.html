We are trying to understand how people perceive object symmetries. We have developed a computer program to identify potential symmetries in a object. Your task to help guide our program by indicating to the program when images of the object are the same (indicating a correct symmetry) or different.

You will be given pairs of images. Place you mouse on top and out of the image box to see the different image. Please indicate if you think the two images are identical ("None"), or if you can detect the two images as being different ("Clearly different"). Select "Slight" if you see a difference but you don't think it is semantically important, and "Some" if you see a small difference that can affect the way the object is used (examples are reflected letters and handles on the other side) Select "N/A" if there is something wrong and you can't see any images.
